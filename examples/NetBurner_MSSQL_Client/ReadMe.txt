NetBurner MSSQL Client — Example Firmware
=========================================

Reference SQL browser firmware for NetBurner modules.

Full documentation:
  ../../README.md
  ../../docs/getting-started.md

Quick build and load:

  cd examples/NetBurner_MSSQL_Client
  make PLATFORM=NANO54415
  make PLATFORM=NANO54415 load DEVIP=<device-ip>

Web UI: http://<device-ip>/
Console: http://<device-ip>/console.html
